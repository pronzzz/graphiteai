# Contributing to GraphiteAI

Thank you for your interest in contributing to GraphiteAI! We welcome contributions from the community to help improve this project.

## How to Contribute

1. **Fork the Repository**: click the "Fork" button at the top right of this page.
2. **Clone Your Fork**:

    ```bash
    git clone https://github.com/YOUR_USERNAME/graphiteai.git
    cd graphiteai
    ```

3. **Create a Branch**:

    ```bash
    git checkout -b feature/amazing-feature
    ```

4. **Make Your Changes**: Implement your feature or fix.
5. **Commit Your Changes**:

    ```bash
    git commit -m "Add some amazing feature"
    ```

6. **Push to Your Branch**:

    ```bash
    git push origin feature/amazing-feature
    ```

7. **Open a Pull Request**: Go to the original repository and click "Compare & pull request".

## Code Style

- **Python**: We follow [PEP 8](https://www.python.org/dev/peps/pep-0008/) guidelines.
- **JavaScript**: Use consistent indentation and semicolons.
- **Documentation**: Ensure any new features are well-documented in the README or code comments.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the [Issues](https://github.com/pronzzz/graphiteai/issues) tab. Provide as much detail as possible.
